# 🚀 Stan's Robot Shop - DevOps Project

## 📌 About This Project
This is a DevOps project where I deployed the Stan's Robot Shop application on AWS EC2.  
I worked on setting up the server, configuring services, and making the application fully functional.

---

## 🛠️ Technologies Used
- AWS EC2
- Linux
- Nginx
- MongoDB
- Git & GitHub

---

## 🚀 What I Did
- Deployed the application on EC2
- Configured Nginx as reverse proxy
- Connected MongoDB database
- Made sure the payment flow is working
- Managed services using systemd
